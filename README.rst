============================
Modular cartoons
============================

-----------
Description
-----------
This is just a basic set of SVG primitives, required to build simple, but good-looking cartoons. 

Initially created by `AterLeo`_

.. _AterLeo: http://leo-tvorit.lj.ru

--------
Features
--------
- modular design
- basic shapes and various mimics
- complete characters
